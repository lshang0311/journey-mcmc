# journey-mcmc

    A MCMC (Markov chain Monte Carlo) based event sequence generator

     * Simulation details
        > Event sequence
        > Time stamps
        > Event attriubtes

     * Technology stack
        > R/Python/Scala/Java
        > Akka/RxJava
        > Cassandra

     * Requirements
        > Generic for any event based analysis (Retail, Banking, Telcos)  
        > Configurable for data size, shape, and relationship
        > ...
